# VPC-SUBNET-CREATION

A subnet is a logical subdivision of a VPC that allows you to organize and 
isolate resources within different Availability Zones.

## Before You Begin
- AWS Account
- Existing VPC

## Parameter	         Value
- VPC CIDR  	        192.168.0.0/16
- Subnet Name 	      my--subnet01
- Availability Zone  	ap-south-1a
- Subnet CIDR	        192.168.1.0/24

## Verification

- Go to VPC → Subnets.
- Verify that the new subnet appears in the subnet list.
- Confirm the CIDR block, Availability Zone, and associated VPC
